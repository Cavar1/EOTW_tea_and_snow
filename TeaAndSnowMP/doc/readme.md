# 奈茶的雪 - 微信小程序期末作业项目说明文档

## 一、主题

本项目是一款面向奶茶/烘焙门店的微信小程序——**奈茶的雪**，模拟了点单、下单、查看订单、会员信息等核心场景，运用清新色调以及手作风格画面，配合轻量后端接口，提供简洁流畅的点单与订单管理体验。

## 四、核心功能

1. **首页浏览**：展示店铺横幅广告与快捷点单入口。
2. **茶点选购**：按分类浏览茶点，查看商品详情，加入购物车，调整数量，提交订单。
3. **订单管理**：查看历史茶单列表、茶单详情，支持待支付订单的支付操作。
4. **会员信息**：展示并编辑用户头像、昵称、会员等级等个人信息。
5. **自定义导航与 Tab**：使用自定义导航栏和底部 Tab 切换，提升页面体验。
